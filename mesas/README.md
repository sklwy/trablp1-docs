## Modelo (Object)

### Mesas

##### Rotas

*insert*: `"/api/mesas"` [`POST`]<br>
*listMesas*: `"/api/mesas/list` [`GET`]<br>

##### Campos 
| **CAMPOS** | **TIPO** | **TAMANHO** | **TRADUÇÃO** |<br>

| `mesEspCod` | String | 15 | Número Mesa |<br>
